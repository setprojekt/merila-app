❌ NE PRENESI TEH DATOTEK/MAP:
------------------------------

- vendor/ (se namesti z composer install)
- node_modules/ (se namesti z npm install)
- .git/ (če ne uporabljate git na strežniku)
- .env (ustvarite na strežniku)
- storage/logs/*.log
- storage/framework/cache/*
- storage/framework/sessions/*
- storage/framework/views/*
- bootstrap/cache/*.php (razen .gitignore)
- public/build/
- public/hot
- public/storage
- merila_db (lokalna baza, če obstaja)

⚠️ POMEMBNO:
-----------

1. Storage mapa: Prenesi samo strukturo map, ne vsebine
   - storage/app/
   - storage/framework/cache/
   - storage/framework/sessions/
   - storage/framework/views/
   - storage/logs/
   (v vsaki mapi pustite .gitignore datoteko, če obstaja)

2. Bootstrap cache: Prenesi samo .gitignore, ne cache datotek

3. Public mapa: Prenesi vse RAZEN:
   - build/ (se generira z npm run build)
   - hot (development datoteka)
   - storage/ (se ustvari z php artisan storage:link)

4. Po prenosu na strežniku:
   - composer install --no-dev --optimize-autoloader
   - npm install
   - npm run build
   - cp .env.example .env
   - php artisan key:generate
   - php artisan storage:link
   - Nastavi pravice: chmod -R 775 storage bootstrap/cache
